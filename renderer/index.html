Keep syncing 24/7

Your POS data is on a network folder. To keep sending it even when no one is signed in to this PC, the connector needs this computer's Windows sign-in. It's stored only on this PC and used just to run in the background.

Don't want to store it? You can skip — the connector will then sync only while someone is signed in to this PC (fine for an always-on register PC).

  2. Install the connector

  5. Keep the PC awake for 24/7 sync
